Features implémentées :

- Un personnage qui peut se déplacer dans toutes les directions

- Une caméra top-down qui suit le joueur

- Mécanique de déplacement rapide (dash) avec temps de recharge

- HUD basique

- Un système de points de vie et de mort

- Ennemi avec une IA simple (idle -> chase -> attack)

- Une feature de combo (X attaques qui peuvent s'enchaîner)

- Écran de défaite placeholder (pas de boucle de jeu)

Bugs connus :
La distance parcourue lors d'un dash semble être différente parfois
Les IA restent parfois en state chase lorsqu'ils sont déjà sur le joueur

Durée sur le test : 23h~23h30
